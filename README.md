多人多群在线聊天室

数据存储于服务端

可更换主题色

`React hook` + `NodeJS` + `Typescript` + `Redux` + `MySQL`


用户和群员两种不同的身份：
- 用户：超级管理员、高级管理员、管理员、普通用户
- 群员：群主、管理员、普通群员
